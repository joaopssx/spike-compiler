# Spike

Spike e um compilador educacional de Portugol escrito em C++.

O README em ingles esta em `README.md`.

No momento, o projeto le um arquivo `.por`, executa a analise lexica e imprime os tokens encontrados.

## Compilacao

```bash
cmake -S . -B build
cmake --build build
```

## Uso

```bash
./build/spike tokens examples/exemplo.por
```
