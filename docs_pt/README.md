# Documentacao em Portugues

Esta pasta contem a documentacao do Spike traduzida para portugues.

## Conteudo

- `docs_pt/grammar.md` - gramatica da linguagem e precedencia
- `docs_pt/architecture.md` - arquitetura do compilador e pipeline
- `docs_pt/tutorial.md` - tutorial rapido com exemplos
- `docs_pt/roadmap.md` - roadmap publico do projeto
- `docs_pt/contributing.md` - guia de contribuicao
- `docs_pt/changelog.md` - historico detalhado de mudancas

Se voce preferir a versao em ingles, consulte a pasta `docs/`.
