Tutorial

Build, inspect and run a Portugol program.

Spike exposes a small but practical CLI. Start with `tokens`, confirm the AST with `ast`, then build or run the final program.

$ spike tokens examples/hello.por
$ spike ast examples/hello.por
$ spike run examples/hello.por

1. Build

`cmake -S . -B build` and `cmake --build build` generate the Spike executable.

2. Inspect

Use `spike tokens` and `spike ast` to understand what the compiler sees internally.

3. Run

`spike run programa.por` emits C, builds the native binary and executes it.

For a complete walkthrough, open `docs/tutorial.md` in the repository.
