Learn compilers with Spike

Modern Portugol, native binaries, clean C++.

Spike is a modern Portugol compiler written in C++ designed for learning, experimentation and performance. It lexes, parses, validates and transpiles Portugol into C before calling a native compiler.

Recursive descent parser

Spike keeps the parser explicit and readable, which makes the grammar easier to evolve and easier to teach.

AST-first architecture

The compiler builds an internal tree before semantic checks and code generation, keeping each stage focused.

Native build flow

Spike emits C and can immediately call `gcc`, `clang` or another compatible compiler to generate an executable.
