# Styles

Spike will support multiple Portugol styles in the future.

At this stage, this is still under development.

The planned styles for now are:

- `Default`
- `Iostream`
- `Braces`

Goal of this initial base:

- prepare the project for syntax variations
- keep the lexer ready for new formats
- avoid changing the parser too early

For now, the compiler only recognizes some of these style tokens in the lexer.
