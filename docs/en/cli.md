# CLI Reference

Main commands:

- `spike build file.por`
- `spike run file.por`
- `spike check file.por`
- `spike tokens file.por`
- `spike ast file.por`
- `spike ir file.por`
- `spike stats file.por`
- `spike explain file.por`
- `spike emit-c file.por`
- `spike trace file.por`
- `spike doctor`
- `spike version`

Useful flags:

- `--time`
- `--trace`
- `--json`
- `--graph`
- `--output <file>`
- `--stdout`
- `--with-spans`
- `--emit-c`
- `--no-build`
- `--cc <compiler>`
- `-o <binary>`
