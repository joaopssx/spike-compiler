# Estilos

O Spike vai suportar multiplos estilos de Portugol no futuro.

Nesta etapa, isso ainda esta em desenvolvimento.

Os estilos planejados por enquanto sao:

- `Default`
- `Iostream`
- `Braces`

Objetivo desta base inicial:

- preparar o projeto para variacoes de sintaxe
- manter o lexer pronto para novos formatos
- evitar mudar o parser cedo demais

Por enquanto, o compilador apenas reconhece alguns tokens desses estilos no lexer.
