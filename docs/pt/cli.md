# Referencia da CLI

Comandos principais:

- `spike build arquivo.por`
- `spike run arquivo.por`
- `spike check arquivo.por`
- `spike tokens arquivo.por`
- `spike ast arquivo.por`
- `spike ir arquivo.por`
- `spike stats arquivo.por`
- `spike explain arquivo.por`
- `spike emit-c arquivo.por`
- `spike trace arquivo.por`
- `spike doctor`
- `spike version`

Flags uteis:

- `--time`
- `--trace`
- `--json`
- `--graph`
- `--output <arquivo>`
- `--stdout`
- `--with-spans`
- `--emit-c`
- `--no-build`
- `--cc <compilador>`
- `-o <binario>`
